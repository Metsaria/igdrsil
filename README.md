# Igdrsil — Releases

**AI for confidential work, on your infrastructure.**

Igdrsil gives your team one place to work with Projects, Tasks, documents, Tapio and Dupin. Your documents and conversations stay on your organisation's systems.

## Start here

- [`documentation/01-user-documentation/`](documentation/01-user-documentation/) — how to use Igdrsil.
- [`documentation/02-developer-extension-documentation/`](documentation/02-developer-extension-documentation/) — the only developer material included here: how to create customer-owned extensions.
- [`releases/`](releases/) — release notes and update information.
- [`binaries/`](binaries/) — install packages, when published.

## What is not here

This release package does not contain product strategy, architecture notes, source code, internal plans, customer data, telemetry, support exports or build instructions.

## Product promise

- **Data stays with you.** Igdrsil is not a hosted AI service behind the scenes.
- **Built for non-technical work.** Everyday use should not require developer tools.
- **Transparent updates.** Release notes describe only customer-visible changes and required customer actions.
