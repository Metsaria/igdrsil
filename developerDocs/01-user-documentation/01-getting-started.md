# Getting Started

## What you need

- An Igdrsil account created by your administrator.
- The Igdrsil desktop client or browser access provided by your firm.
- A connection to your firm's Igdrsil server.

The server runs on Linux or macOS. The client runs on Linux, macOS, and Windows.

## First login

1. Open Igdrsil.
2. Sign in with the username and temporary password your administrator gave you.
3. Change the temporary password if prompted.
4. Open the default Igdrsil Project.
5. Type a question to Tapio in the chat.

Good first questions:

- "What can I do here?"
- "How do I add documents?"
- "How do I check a document with Dupin?"

## Your first useful task

1. Create or open a Project for the matter you are working on.
2. Create a Task for the concrete work item.
3. Upload a document into the Task.
4. Ask Tapio what agent should handle the task.
5. If citation checking is needed, add Dupin and ask it to run a citation review.

## If something does not work

Ask your firm administrator first. Igdrsil is sovereign by design: the vendor cannot log in to your firm's server or inspect your data remotely.
