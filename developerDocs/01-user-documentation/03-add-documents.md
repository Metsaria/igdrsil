# Add Documents

Igdrsil can only answer from documents it is allowed to read. A document collection is called a **Corpus Source**.

## What a Corpus Source is

A Corpus Source is a folder of firm documents registered with Igdrsil. The documents stay where they are. Igdrsil reads them and builds a local search index so agents can find relevant passages.

Examples:

- A folder of previous case files.
- A folder of approved templates.
- A folder of saved decisions or reference material.

## Add a Corpus Source

1. Open **Corpus Sources**.
2. Click **Add Source**.
3. Select the folder path your server can access.
4. Choose the document type.
5. Enter the lawful basis or internal reason your firm is allowed to process the material.
6. Give the source a clear name.
7. Click **Register**.

Your administrator may need to perform this step if you do not have permission.

## Indexing

After registration, Igdrsil indexes the source. During indexing, the source shows progress. When it is ready, agents can use it.

Small folders may finish in minutes. Large folders can take longer. You can continue working while indexing runs.

## Keep sources current

When new files are added to a registered folder, Igdrsil indexes them. When files are removed, Igdrsil removes them from search results. Source files are not modified by Igdrsil.
